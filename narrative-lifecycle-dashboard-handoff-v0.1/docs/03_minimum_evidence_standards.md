# 03 Minimum Evidence Standards

## Evidence Strength

- **E0 Invalid**: rumor, unverifiable screenshot, no source.
- **E1 Weak**: single news, opinion, short-term price move.
- **E2 Valid**: authoritative source, clear event.
- **E3 Strong**: multiple sources, continuity, market impact.
- **E4 Closed-loop**: perception, capital, and reality mutually reinforce.

## Stage Minimums

| Stage | Minimum Evidence |
|---|---|
| S0 | Real weak signal from credible source |
| S1 | Attention breaks out beyond the original professional circle |
| S2 | Explanatory framework and future hypothesis |
| S3 | Stable label and initial stock/industry mapping |
| S4 | Capital validation through leader, breadth, volume, persistence |
| S5 | Pricing adoption: valuation reframing and financial translation |
| S6 | Hard reality evidence: orders, revenue, customers, regulatory/clinical/product validation |
| S7A | Continuous validation, long-term allocation, stable business model |
| S7B | Good-news fatigue, overcrowding, reality below expectations |
| S7C | New branch, new label, new expectation gap |

## Critical Rule

Missing data does not mean negative evidence. It lowers confidence and caps stage judgment.
