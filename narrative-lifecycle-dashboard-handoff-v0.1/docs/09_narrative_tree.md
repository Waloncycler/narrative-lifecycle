# 09 Narrative Tree

## Purpose

A market narrative is a tree, not a line.

## Levels

- **L0 Macro Theme**: broad long-term background.
- **L1 Parent Narrative**: main tradable theme.
- **L2 Branch Narrative**: independently evolving sub-theme.
- **L3 Sub-branch / Scenario**: application, technology route, supply-chain link.
- **L4 Asset Mapping**: company, product, project, index, ETF.
- **L5 Evidence Node**: event or data point.

## Critical Rule

```text
Parent Stage ≠ max(Branch Stage)
```

Parent stage is determined by:

- parent evidence
- maturity of core branches
- branch coverage
- number of validating branches
- inter-branch evidence network
- whether capital and pricing generalize from branch to parent

## Branch Fields

- branch_importance
- branch_coverage_score
- branch_maturity_gap
- branch_independence
- branch_feedback_to_parent
- parent_lift_potential

## Branch Independence

A branch can become a new parent narrative if it has:

- independent label
- independent stock pool
- independent event stream
- independent pricing logic
- independent capital feedback
- independent failure signals
