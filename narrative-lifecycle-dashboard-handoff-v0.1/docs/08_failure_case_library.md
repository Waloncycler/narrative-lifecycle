# 08 Failure Case Library

## Purpose

Failure cases prevent the system from learning only successful narratives and becoming over-optimistic.

## Failure Types

1. **Attention Trap**: high attention without explanation.
2. **Narrative Trap**: grand logic without stable label.
3. **Label Trap**: clear label without capital validation.
4. **Trading Trap**: strong trading action without pricing adoption.
5. **Expectation Trap**: pricing adoption without reality delivery.
6. **Validation Trap**: reality appears but below priced expectations.
7. **Branch Pollution Trap**: branch validation mistaken as parent validation.
8. **Overcrowding Trap**: story may be true, but expectation and valuation are full.
