# 14 Glossary

- **Narrative**: a market explanation linking events, future expectations, and tradable objects.
- **Perception / 名**: attention, framing, label, interpretation.
- **Capital Confirmation / 资**: funding or market price validation.
- **Pricing Adoption / 定价**: valuation and financial translation.
- **Reality Validation / 实**: hard evidence from real-world execution.
- **Feedback / 势**: self-reinforcing loop between perception, capital, and reality.
- **Friction / 阻力**: obstacles that block stage transition.
- **Stage Gate**: minimum evidence rule for each stage.
- **Narrative Delta**: material difference between current and previous narrative state.
- **Branch Coverage**: how much branch evidence represents the parent narrative.
- **Golden Case**: baseline case that implementation must reproduce.
- **Failure Case**: historical trap used to calibrate false positives.
- **Early Radar**: system that tracks S1-S4 and S7C branch opportunities.
