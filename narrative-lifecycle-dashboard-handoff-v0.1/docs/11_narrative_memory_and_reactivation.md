# 11 Narrative Memory and Reactivation

## Purpose

Most opportunities are not completely new topics. They are old themes with new evidence, new stages, new branches, reality catch-up, or expectation reset.

The system is a **Narrative State Change Detector**, not just a New Topic Scanner.

## Narrative Memory Bank

Stores:

- first_seen_date
- last_active_date
- historical_stage_path
- previous_peak_stage
- previous_failed_transition
- previous_failure_reason
- previous_missing_evidence
- previous_friction_points
- previous_branch_structure
- is_failure_case
- memory_confidence

## Reactivation Types

- new_topic
- dormant_signal_reactivation
- stage_reactivation
- branch_mutation
- reality_catch_up
- expectation_reset
- repeated_old_story
- failed_revival_attempt
- noise

## Narrative Delta Score

Measures what is materially different from the previous narrative state.

Components:

- New Evidence Quality
- Stage Gate Impact
- Missing Evidence Filled
- Branch Mutation Strength
- Expectation Reset
- Data Confidence

## Rules

1. Old theme reappearing does not equal new opportunity.
2. Repeated old logic does not enter Early Radar.
3. Old failed themes require higher reactivation evidence.
4. New branches are modeled independently.
5. Reality catch-up can reactivate old themes.
