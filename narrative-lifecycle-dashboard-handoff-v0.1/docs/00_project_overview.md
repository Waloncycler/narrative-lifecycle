# 00 Project Overview

## Goal

Build a Narrative Lifecycle Dashboard that helps researchers classify market narratives, identify early opportunities, monitor validation, and detect narrative traps.

The system is based on:

- 名 / Perception
- 资 / Capital Confirmation
- 定价 / Pricing Adoption
- 实 / Reality Validation
- 势 / Feedback Momentum
- 阻力 / Friction
- 数据置信度 / Data Confidence

## Main Question

How does a market narrative move from weak signal to shared perception, from label to capital validation, from capital validation to pricing adoption, and from pricing adoption to reality validation?

## Output Types

- Dashboard Card
- Early Radar Candidate
- Narrative Tree
- Evidence Table
- Failure Case Record
- Weekly Narrative Brief

## Non-goals

- No direct buy/sell advice.
- No automated trading.
- No black-box LLM-only scoring.
