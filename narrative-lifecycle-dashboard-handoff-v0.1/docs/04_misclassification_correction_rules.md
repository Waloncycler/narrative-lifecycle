# 04 Misclassification Correction Rules

## Core Corrections

1. 热度不等于叙事。Attention is not narrative formation.
2. 政策命名不等于资金共识。Policy naming is not capital confirmation.
3. 资金异动不等于定价采纳。Price action is not pricing adoption.
4. 券商报告不等于现实验证。Research reports are not hard reality evidence.
5. 战略合作不等于订单。Strategic partnership is not order validation.
6. 订单不等于收入。Orders are not revenue.
7. 收入不等于利润。Revenue is not profit.
8. 分支验证不等于主叙事验证。Branch validation is not parent validation.
9. 个股验证不等于板块验证。Single-company validation is not sector validation.
10. 高热度不等于高赔率。High heat is not high payoff.
11. S6 不等于安全。Reality validation is not safety.

## Stage Gate Corrections

- No stable label → max S2.
- No capital confirmation → max S3.
- No pricing adoption → max S4.
- No hard reality evidence → max S5.
- Only branch evidence → do not automatically upgrade parent.
