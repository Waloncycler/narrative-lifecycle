# 12 MVP Scope and Product Shape

## MVP Definition

A semi-automatic research system that uses structured evidence, explicit rules, and LLM-generated reasoning to produce narrative lifecycle dashboard cards.

## MVP Modules

1. Topic Library
2. Narrative Memory Bank
3. Evidence Table
4. Stage Classifier
5. Scoring Engine
6. Narrative Tree
7. Dashboard Card
8. Early Radar
9. Failure Case Library

## MVP Does Not Include

- Automated trading
- Buy/sell signals
- Full-market automatic scanning
- Real-time high-frequency analysis
- Full LLM-only stage judgment
- Complex UI animations

## First Product Shape

Recommended path:

1. YAML/Markdown + rule engine.
2. Semi-automatic prototype.
3. Web dashboard after rules and cases stabilize.
